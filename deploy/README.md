# Deployment Assets

Kubernetes packaging for the FalseFlag stack, operator, and CRDs.

- `crds/**` — generated CustomResourceDefinitions, regenerated by
  `make generate` from `operator/api/v1alpha1/`. Apply directly with
  `kubectl apply -f deploy/crds/` or let the Helm chart install them.
- `helm/falseflag-operator/**` — Helm chart for the full stack
  (Postgres, API, proxy, MCP, dashboard). Run `helm install falseflag
  deploy/helm/falseflag-operator -n falseflag-system
  --create-namespace`. The operator is opt-in with
  `--set operator.enabled=true`. CRDs ship in `crds/` per Helm
  convention.
- `kustomize/**` — Kustomize overlays. `kubectl apply -k
  deploy/kustomize/overlays/dev` installs the namespace, CRDs, RBAC,
  ConfigMap, and Deployment in one shot. The `dev` overlay points the
  operator at `host.docker.internal:8090` for kind-based local
  development.
- `samples/**` — example CRs covering the full Project / Environment /
  Segment / RolloutPolicy / Flag / FlagBinding / FlagSnapshot graph.
  Apply after the operator is running: `kubectl apply -k
  deploy/samples`.

## Quickstart with kind

```bash
make up                          # API + Postgres + dashboard via compose
kind create cluster
kubectl apply -k deploy/kustomize/overlays/dev
kubectl apply -k deploy/samples
kubectl get projects.falseflag.dev -A
```

See `operator/README.md` for the full operator usage guide.
